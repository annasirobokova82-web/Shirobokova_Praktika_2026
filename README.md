\## Описание проекта



Приложение для учёта партнёров компании, с расчётом скидок на основе объёма продаж.



\## Технологии



\- C# / .NET 8.0

\- WPF

\- Entity Framework Core

\- PostgreSQL

\- MSTest



\## Структура проекта



\- `/Database` - скрипты базы данных

\- `/Project\_ShirobokovaPartnerApp` - исходный код

\- `/ShirobokovaPartnerApp.exe` - собранный исполняемый файл

\- `/Screenshot` - скриншоты 

\- `/Широбокова\_Отчет` - отчет

\## Запуск



1\. Установить PostgreSQL

2\. Выполнить скрипт Database/create\_database.sql

3\. Открыть решение в Visual Studio

4\. Нажать F5
